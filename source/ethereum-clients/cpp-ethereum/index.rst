Quick Start
================================================================================

- The Github project for the Expanse C++ client is https://github.com/expanse-org/cpp-expanse/
- As of right now, though (only for the v1.3.0 release), the canonical mainline is at https://github.com/bobsummerwill/cpp-expanse/tree/merge_repos
- Prior to the v1.3.0 release and for several months before, the canonical mainline was at https://github.com/expanse-org/webthree-umbrella
- If you just want to install binaries then head to :ref:`Installing binaries`.
- If you want to build from source then head to :ref:`Building from source`.
- You can chat with the community and developers at `cpp-expanse-gitter`_
- The developers have their in-depth conversations at `cpp-expanse-development-gitter`_
- Please log any issues using the `Github issue tracker <http://github.com/expanse-org/webthree-umbrella/issues>`_.
- cpp-expanse is extremely portable and is used on a :ref:`very broad range of platforms <cpp-expanse-portability>`.

.. _cpp-expanse-gitter: https://gitter.im/expanse/cpp-expanse
.. _cpp-expanse-development-gitter: https://gitter.im/expanse/cpp-expanse-development
